HexKeyLab Sample Analyzer

Fetches every chromatic note in each instrument's range from the soundfont CDN, decodes it with Web Audio (identical to HexKeyLab's runtime), runs prepareLoop, and displays the results so you can decide which to include.

Missing notes (404s) are silently skipped. After analysis, use the checkboxes to pick which samples to keep, then copy the output into HexKeyLab.html.

Instrument ranges (editable)

Each instrument lists its baseUrl, note range (chromatic, inclusive), and optional gateOpts overrides for tuning the unified analyzer pipeline (corrThreshold, slopeStepThreshold, etc.). Ranges go from lowOctave to highOctave; any notes returning 404 are skipped automatically.
