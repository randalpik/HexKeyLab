JI chord player (RPN sequential retune)

  
One voice per MIDI channel: each tone claims its own channel, that channel is fine-tuned (RPN 0001) to the tone's offset, then all note-ons fire together as a block chord. Distinct channels are the only way to hold distinct tunings at once — a single channel re-applies its current fine-tune to every voice at each note-on. 16 voices max, oldest stolen after that. Needs the SP-250 set to receive multiple channels (Omni On or multi-timbral) — if only some notes sound, check its MIDI receive mode.

Tuning-apply test (channel rotation)

  
Plays the root note 12 times, alternating its fine-tune 0¢ / +50¢, with the RPN sent immediately before each note-on (no gap — like HKL). Same MIDI note in both modes, so anything you hear is purely the fine-tune applying or not. Same channel keeps it on ch1 (the in-tune LIFO case); Rotating moves to the next channel each note (the FIFO case). If Same alternates cleanly but Rotating doesn't, switching channels with an immediate note-on races the fine-tune.
